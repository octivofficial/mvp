# 마인크래프트 클라이언트 접속 가이드

## 서버 정보
- **주소**: localhost:25565 (또는 127.0.0.1:25565)
- **버전**: Minecraft 1.21.1
- **모드**: 오프라인 (정품 인증 불필요)
- **게임모드**: 서바이벌

---

## 해결 방법 1: 클라이언트 버전을 1.21.1로 변경 (권장)

### 단계:
1. **마인크래프트 런처 실행**
2. **설치 탭** 클릭
3. **새로 만들기** 클릭
4. **버전 선택**: 드롭다운에서 `release 1.21.1` 선택
5. **만들기** 클릭
6. **플레이** 탭으로 돌아가서 방금 만든 프로필 선택
7. **플레이** 클릭

### 접속:
1. 게임 실행 후 **멀티플레이어** 클릭
2. **서버 추가** 클릭
3. 서버 이름: `Octiv AI Sandbox`
4. 서버 주소: `localhost` (또는 `127.0.0.1`)
5. **완료** → 서버 선택 → **서버 접속**

---

## 해결 방법 2: ViaVersion 플러그인 사용 (다른 버전 허용)

서버에 ViaVersion 플러그인을 설치하면 다양한 클라이언트 버전을 허용할 수 있습니다.

### 단계:
1. ViaVersion 다운로드: https://www.spigotmc.org/resources/viaversion.19254/
2. `server/plugins/` 폴더에 jar 파일 복사
3. 서버 재시작

**주의**: 이 방법은 서버 재시작이 필요하며, 현재 실행 중인 AI 에이전트가 연결이 끊어집니다.

---

## 해결 방법 3: 서버 버전 업그레이드 (비권장)

현재 서버를 최신 버전으로 업그레이드할 수 있지만, AI 에이전트 호환성 문제가 발생할 수 있습니다.

---

## 현재 AI 에이전트 활동

접속하면 다음 AI 에이전트들을 볼 수 있습니다:
- **Octiv_builder-01** ~ **Octiv_builder-05**: 나무 수집 중
- **위치**: 스폰 근처 (X: 40~60, Z: 25~35)
- **미션**: AC-1 나무 수집 (16개 목표)

---

## 문제 해결

### "Connection refused" 에러
- 서버가 실행 중인지 확인
- 주소가 `localhost` 또는 `127.0.0.1`인지 확인

### "Outdated server" 또는 "Outdated client" 에러
- 클라이언트 버전을 정확히 1.21.1로 설정
- 런처에서 버전 확인: 설치 탭 → 프로필 편집 → 버전 확인

### 접속 후 "You are not whitelisted" 에러
- 현재 화이트리스트는 비활성화되어 있으므로 이 에러는 발생하지 않아야 함
- 발생 시 서버 콘솔에서: `/whitelist off`
