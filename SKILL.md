# Octiv — AI Agent Sandbox (SKILL.md)

## 🎯 최종 목표
PaperMC 1.21.1 마인크래프트 서버(localhost:25565)에서 AI 봇을 자율 운용한다.

## 핵심 스킬 목록
| 스킬 | 용도 |
|------|------|
| `mcporter` | Minecraft 봇 생성/제어 (mineflayer 기반) |
| `coding-agent` | 봇 코드 작성/수정 자동화 |
| `github` | 작업 기록, PR 자동화 |
| `health-monitor` | 서버/에이전트 상태 모니터링 |
| `dev-tool-belt` | 개발 도구 (lint, test, git) |
| `strategy-engine` | 목표 분해 및 전략 수립 |
| `mcp-tool-bridge` | 외부 MCP 서버 연결 |
| `automated-debugging` | 에러 자동 진단/수정 |
| `gemini` | 빠른 Q&A, 요약 |
| `weather` | 날씨 정보 (게임 내 일정 계획용) |
| `notion` | 진행 상황 문서화 |

## 서버 접속 정보
- **Minecraft**: `localhost:25565` (offline-mode)
- **RCON**: `localhost:25575` / pw: `octiv_rcon_2026`

## 에이전트 운용 원칙 (眞善美孝永)
1. **眞** — 게임 상태를 정확히 읽고 판단
2. **善** — 서버 안정성 최우선, 안전한 행동만
3. **美** — 체계적인 건축, 정돈된 수납
4. **孝** — 사령관에게 진행 상황 주기적 보고
5. **永** — 지속 가능한 자원 관리, 장기 베이스 구축
